#### **Overview**

Formats a number as text. Group sizing and separator and other locale-specific configurations are based on the pattern data.

#### **Usage**

```
{{ value | numberFormat }}

```


#### **Input Value**

| Parameter |  Type  |         Description         |
| :-------: | :----: | :-------------------------: |
|   value   | number | The number to be formatted. |
